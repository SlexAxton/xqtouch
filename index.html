xqtouch by SlexAxton

A plugin for xui that exposes the exact api needed to run jqTouch.

This script attempts to patch the jquery api used in the jqTouch application onto the xui platform. Some changes had to be made to the xui library directly, but they are very general additions and/or bugfixes to the library and could possibly be pulled into the master repository. Currently my branch of XUI is at the following address: Alot of the code for this project is taken directly from jQuery 1.3.2 in order to maintain compatibility. I have tried to mark the functions that are copy and pasted from the jquery library as best as possible. jQuery (by John Resig | http://ejohn.org) is available at: http://www.jquery.com/

Dependencies

xui jqtouch

Install

1) Include (preferably my branch of) xui. 2) Include xqtouch 3) Include jqtouch
